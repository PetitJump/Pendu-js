# Jeu du Pendu

Un petit jeu du pendu réalisé en **HTML** et **JavaScript**. Le joueur doit deviner le mot caché en proposant des lettres. Chaque erreur fait avancer le pendu sur l'image, et le joueur a un nombre limité de vies.

## Fonctionnalités

- Deviner un mot lettre par lettre.
- Affichage des lettres déjà utilisées.
- Affichage du mot partiellement trouvé.
- Gestion des vies restantes.
- Images représentant l'évolution du pendu.

## Comment jouer

1. Ouvrir le fichier `index.html` dans votre navigateur.
2. Saisir une lettre dans le champ prévu et cliquer sur "submit".
3. Si la lettre est correcte, elle apparaîtra dans le mot à deviner.
4. Si la lettre est incorrecte, vous perdrez une vie et le pendu évoluera.
5. Le jeu continue jusqu'à ce que :
   - Vous trouviez toutes les lettres du mot → **Victoire !**
   - Vous n'ayez plus de vies → **Défaite !**
